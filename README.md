# Periodic-Table-Explorer
Periodic Table Explorer

## Usage/Examples

```javascript
$> git clone https://github.com/Len-L/Periodic-Table-Explorer.git
$> cd Periodic-Table-Explorer
$> pip install -r requirements.txt
$> python3 main.py
```
## Features
- Search for elements by name/symbol/number
